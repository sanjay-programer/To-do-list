# 📝 To-Do List Web App

This is a simple To-Do List web application built using **HTML, CSS, and JavaScript** during my early learning phase of web development.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and clean UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)

## 📚 What I Learned
- DOM manipulation using JavaScript
- Handling user input
- Basic event handling
- Structuring a simple frontend project
